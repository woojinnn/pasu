제안 · Editor 진입 시 nav lock-collapsed

Editor의 Block palette는 좌측 18% (≈260px @1440). nav overlay가 256px이라 거의 완전히 겹침 — hover로 펼치면 palette 잡을 수 없음. 그래서 Editor 페이지에서는 nav가 collapsed로 잠김: rail은 64px 그대로 보이고 hover/focus overlay 비활성. 메뉴 이동은 키보드(⌘+1~5)나 rail 아이콘 직접 클릭으로 가능. nav 펼치려면 명시적 ⌘+B 토글.

대안 · overlay가 palette를 덮음 (택안)

nav overlay가 palette 위를 덮되, palette 영역 (좌측 260px)에 마우스가 들어가면 overlay 자동 접힘. 펼친 nav에서 항목 클릭하려면 잠시 멈춰야 함 (200ms hover-intent). 더 자연스럽지만 palette 잡을 때 nav가 의도와 다르게 펼쳐질 위험. Editor 빈도 높으면 lock-collapsed 추천.
