# Signature Fixtures

These EIP-712 fixtures are synthetic test shapes derived from the EIP-2612
permit specification and the Permit2 ABI/type structure.

Sentinel addresses such as `0xaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa`,
`0x1111111111111111111111111111111111111111`, and
`0x9999999999999999999999999999999999999999` are test-only placeholders and
are not real chain data.
